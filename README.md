# TechNews — Portal de Notícias

Landing page de um portal de notícias de tecnologia, com tema escuro e layout em grid. Projeto desenvolvido durante a formação **Full Stack** da [Rocketseat](https://www.rocketseat.com.br/), como prática de HTML semântico e CSS moderno (Grid, Nesting e variáveis).

![Preview do projeto](./assets/screenshot.png)

## Funcionalidades

- Seção de destaque com notícia principal e cards secundários
- Bloco "Mais lidas da semana" com tags de categoria
- Seção de Inteligência Artificial com artigos detalhados
- Barra lateral com espaço para anúncios e recomendações
- Navegação com categorias de tecnologia

## Tecnologias

- HTML5
- CSS3 (Grid, Nesting, variáveis CSS)
- Google Fonts (Archivo)

## Estrutura de arquivos

```
.
├── index.html
├── styles/
│   ├── index.css       # importa os demais arquivos de estilo
│   ├── global.css      # reset, variáveis e estilos base
│   ├── header.css      # navegação primária e secundária
│   ├── sections.css    # estilos das seções
│   └── utility.css     # classes utilitárias
└── assets/
```

## Como executar

Basta abrir o arquivo `index.html` no navegador. Não há dependências ou instalação necessárias.

```bash
git clone <https://github.com/isabellesouzaz/projeto-portal-de-noticias>
cd projeto-portal-de-noticias
```
